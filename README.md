Fidesz által inspirált szortírozó algoritmus.

Az algoritmus leírása: 
    1. lépés:
        Végig haladunk a listán, és megkeressük a legnagyobb(legazdagabb) elemet(réteget).
    2. lépés:
        Újból végig haladunk a listán, és a megtalált maximum helyén lévő értékhez hozzáadjuk az összes többi helyen lévő értéket, mielőtt lenullázzuk azokat. (Tehát a legzadagabb érdekében meglopunk mindenki mást.)
    3. lépés:
        Az ellenzékre vagy Brüsszelre fogjuk a gzadaság hanyatlását.
    Tényleges szortírózás nem történik, ahogy fidesz alatt rendes kormányzás sem, viszont helyette mindenki meg lesz lopva.
